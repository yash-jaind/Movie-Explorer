# CineSearch – Movie Discovery Platform

CineSearch is a responsive movie discovery web application built using JavaScript, HTML5, CSS3, and REST APIs. The platform enables users to search for movies, browse popular titles, explore top-rated content, and view dynamically rendered movie information fetched from external APIs.

## Features

* Movie Search Functionality
* Popular Movies Section
* Top Rated Movies Section
* Dynamic Movie Rendering
* REST API Integration
* Responsive User Interface
* DOM Manipulation using JavaScript
* Fetch API for Asynchronous Requests

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* REST APIs
* Fetch API
